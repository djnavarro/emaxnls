# Coefficents for an Emax regression

Coefficents for an Emax regression

## Usage

``` r
# S3 method for class 'emaxnls'
coef(object, back_transform = FALSE, ...)
```

## Arguments

- object:

  An `emaxnls` object

- back_transform:

  Should log-scaled parameters (logEC50, logHill) be back-transformed to
  original scale?

- ...:

  Ignored

## Value

A vector of coefficients
